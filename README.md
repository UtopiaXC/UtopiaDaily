# Utopia Daily

WIP  

这是一个开发中的项目。旨在进行新闻抓取。支持模块化加载和可视化配置网页。支持AI打标分析。    
灵感来源：[TrendRadar](https://github.com/sansan0/TrendRadar)  

警告：本项目大量使用Gemini等生成式AI。  

警告：目前代码均为测试用途，还没有进行业务逻辑的实现，请不要fork，pr等。
